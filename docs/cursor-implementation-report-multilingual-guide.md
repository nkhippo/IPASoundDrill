# Cursor 実装レポート — 多言語学習ガイド（フェーズ1）

> 作成日: 2026-06-26  
> 対象ブランチ: `main`（`3d2ba14`）  
> 指示書: `docs/cursor-multilingual-guide.md`（Claude 設計）

Claude 側への作業報告用サマリー。

---

## 1. 背景

英語学習者向けに、アプリの導入目的・解決する課題・推奨利用法を多言語で提供する独立タスク。フェーズ1は **en / ja / ko / zh-Hant / zh-Hans** の5言語。

---

## 2. 実施内容

### 2-1. データ（`data/guide.json`）

| 項目 | 内容 |
|------|------|
| 言語 | en, ja, ko, zh-Hant, zh-Hans |
| セクション | 8（welcome … how_to_use） |
| 構造 | 各セクション `title` + `body[]`（全言語で段落数一致） |
| PII | なし（検証済み本文をそのまま格納） |

### 2-2. UI（`index.html`）

| 項目 | 内容 |
|------|------|
| 入口 | 設定モーダル内「ガイド」ボタン |
| 表示 | スクロール可能なガイドモーダル（`#guideModal`） |
| レンダリング | `renderGuide(lang)` — 8セクション固定順、`**強調**` → `<strong>`、`*斜体*` → `<em>` |
| 言語追従 | 開くとき `app_lang` にマッピング（`zh` → `zh-Hans`） |
| 言語ピル | en / ja / ko / 繁體 / 简体 をガイド内で切替可能 |
| フォールバック | 未定義言語は `en` |

### 2-3. i18n（UIラベルのみ）

| キー | 用途 |
|------|------|
| `guide.open` | 設定内ボタン |
| `guide.title` | モーダルタイトル |
| `guide.close` | 閉じるボタン |

`i18n/{en,ja,zh,ko}.json` に追加。本文は `guide.json` から読み込み。
