## Phase 2 開始条件

### 言語別 fontFamily 期待値マップ

| 言語 | UI / body 期待トークン | 見出し期待トークン | IPA 期待トークン |
|---|---|---|---|
| ja | `--font-ui: Noto Sans JP → Noto Sans KR → system-ui` | `--font-serif: Noto Serif JP → Charis SIL` | `--font-ipa: Charis SIL → Doulos SIL` |
| ko | 同上（韓国語 glyph は Noto Sans KR fallback） | 同上 | 同上 |
| zh-Hans | 同上（CJK glyph は Noto Sans JP fallback） | 同上 | 同上 |
| zh-Hant | 同上（CJK glyph は Noto Sans JP fallback） | 同上 | 同上 |
| en | 同上（Latin glyph を共通 UI stack で描画） | 同上 | 同上 |
| fil | 同上（Latin glyph を共通 UI stack で描画） | 同上 | 同上 |

computed `fontFamily` は fallback list を返すため、glyphごとの最終採用fontではなく、上記Design System tokenを含むことをPASS条件とする。

### containerRatio 閾値

- PASS: `0 < element.width / parent.width <= 1.05`
- FAIL: `null`、0以下、または1.05超
- 1.05はsubpixel roundingとborderを許容する上限。overflowは別途 `scrollWidth > clientWidth || scrollHeight > clientHeight` の厳密条件で判定。
